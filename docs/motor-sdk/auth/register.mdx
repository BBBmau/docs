Under the hood, Sonr uses an MPC-based wallet to secure your account and authenticate you. You can create and login to these accounts easily through motor. For more information check out [TODO](#).

## Registering an Account

To create an account, you simply need to provide a recovery password to the [create account method](/docs/motor-sdk/intro.mdx#initialization).. Account creation can take up to three minutes, so entertain your users :).
:::tip

Creating an account automatically authenticates you for the lifetime of the `MotorNode` or `Motoroid` instance. Creating an account automatically authenticates you for the lifetime of the session.

:::

public void createAccount(String password) throws Exception {
