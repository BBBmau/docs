Under the hood, Sonr uses an MPC-based wallet to secure your account and authenticate
you. You can create and login to these accounts easily through motor. For more information
check out [TODO](#).

:::tip

Creating an account bootstraps the Motor `libp2p` node and generates the `MPC Wallet`. This results in the `MotorNode` or `Motoroid` instance being authenticated for the lifetime of the session.

:::

To create an account, you simply need to provide a recovery password to the create account method. Account creation can take up to three minutes, so entertain your users :).

  <TabItem value="go" label="🐹 Go">

:::note

The Golang SDK target requires you to to generate an `aes` key along with the provided pass phrase from the user.
Functionality for key generation can be found within the `Sonr` package `sonr/pkg/crypto/wallet`

:::

```go
func Register(instance *motorNodeImpl, passPhrase string) {
  // create a new key for the account
  key, err := mpc.NewAesKey()
    if err != nil {
      // handle error
    }
    // Note: the generate key should be stored to be used for Authenticaiton.
    // it is recommended to use the platforms `keyring`
    req := mt.CreateAccountRequest{
      Password:  passPhrase,
      AesDscKey: aesKey,
    }

    res, err := instance.CreateAccount(req)
    // if the account is created successfully, you may now interact with the `motor` instance
    m.GetAddress()
}
```

The createAccount method returns a `Future` that resolves to a [`CreateAccountResponse`](/protodocs/motor/v1/response.proto#createaccountresponse) object.

```dart
import 'package:motor_flutter/motor_flutter.dart';

// Create a new account and print the address
final resp = await MotorFlutter.to.createAccount("secure-password-123");
if (resp != null) {
  print(resp.address);
}
```
