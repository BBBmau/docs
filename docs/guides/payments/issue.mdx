Creates a TX in order to deposit the `amount` of SNR into the `recipient` account. A successful transaction will return a [PaymentResponse](/protodocs/motor/v1/response.proto#paymentresponse) and will return null if the transaction fails.

```dart
import 'package:motor_flutter/motor_flutter.dart';

// Issue tokens to a recipient
Future<void> main() async {
  final res = await MotorFlutter.to.deposit('did:snr:abc123', 3.29);
  if (res == null) {
    throw Exception('Failed to deposit SNR');
  }
  print(res); // prints: {txHash: '0x1234567890abcdef', amount: 3.29}
}
```
