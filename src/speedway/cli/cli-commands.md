---
title: Speedway CLI Commands
---
## Registry Commands
### Create a new Account
```bash
speedway registry create
```

### Login to an Account
```bash
speedway registry login
```

## Schema Commands
### Create a new Schema
```bash
speedway schema create
```

### Query a Schema
```bash
speedway schema query
```

## Objects Commands
### Build a new Object
```bash
speedway object build
```

### Get an Object
```bash
speedway object get
```

## Buckets Commands
### Create a new Bucket
```bash
speedway bucket create
```

### Get a Bucket
```bash
speedway bucket get
```

### Get a list of all Buckets (paginated)
```bash
speedway bucket get all
```
